# TryHackMe Cybersecurity Portfolio

Hands-on labs, technical notes, practice scripts, and security exercises completed through **TryHackMe** learning paths.

This repository documents my progression from foundational computing and networking concepts into defensive security, Windows and Active Directory, and Security Operations Center workflows.

My goal is not only to complete rooms, but to document what I learned, explain how the concepts apply to real security work, and build a growing technical portfolio that demonstrates practical cybersecurity skills.
