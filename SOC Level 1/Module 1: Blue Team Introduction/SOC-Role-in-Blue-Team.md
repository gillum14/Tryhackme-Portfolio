# SOC Role in Blue Team

## Learning Objectives
- Understand the purpose of the Blue Team.
- Learn where the SOC fits within an organization's security structure.
- Explore common Blue Team career paths.

## Notes
- **Blue Team** focuses on defending systems by detecting, investigating, and responding to threats.
- Common security teams include:
  - **Red Team:** Offensive security and penetration testing.
  - **Blue Team:** Defensive monitoring and incident response.
  - **GRC:** Governance, Risk, and Compliance.
- A typical **SOC** includes:
  - L1 Analysts
  - L2 Analysts
  - SOC Engineers
  - SOC Manager
- **CIRT/CSIRT** handles major security incidents and digital forensics.
- Specialized roles include:
  - Digital Forensics Analyst
  - Threat Intelligence Analyst
  - Application Security Engineer
  - AI Security Researcher
- **Internal SOC** protects one organization, while an **MSSP** provides security services to multiple clients.
- Typical career progression:
  - SOC L1 → SOC L2 → Engineer, Incident Response, Threat Hunting, Management, or CISO.
