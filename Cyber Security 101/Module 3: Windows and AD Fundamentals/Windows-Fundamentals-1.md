# Windows Fundamentals 1

## Learning Objectives
- Understand the Windows operating system and desktop interface.
- Learn the basics of the NTFS file system and permissions.
- Identify Windows user account types and security features.
- Become familiar with common Windows administration tools.

## Notes
- **Windows 10/11** are the primary desktop operating systems, with Windows Server used in enterprise environments.
- The Windows GUI includes the **Desktop, Start Menu, Taskbar, and Notification Area**.
- **NTFS** is the default Windows file system and supports permissions, encryption, compression, and large files.
- NTFS permissions include **Full Control, Modify, Read & Execute, Read, and Write**.
- **Alternate Data Streams (ADS)** can store hidden data within NTFS files and have legitimate and malicious uses.
- **`C:\Windows`** contains operating system files, while **`System32`** stores critical system utilities.
- Windows uses **Administrator** and **Standard User** accounts to control privileges.
- **User Account Control (UAC)** helps prevent unauthorized system changes by prompting for elevated permissions.
- **Settings** and **Control Panel** are the primary tools for configuring Windows.
- **Task Manager** displays running applications, processes, and system performance.

## Lab
- Explored the Windows desktop and Start Menu.
- Navigated the NTFS file system and viewed file permissions.
- Examined local users and groups.

## Conclusion
This room introduced the core components of the Windows operating system, including the user interface, NTFS, account management, security features, and administrative tools. These fundamentals provide the foundation for managing and securing Windows systems.
