# Windows Fundamentals 2

## Learning Objectives
- Learn Windows administrative and troubleshooting tools.
- Understand system configuration, monitoring, and management.
- Become familiar with essential Windows command-line utilities.

## Notes
- **MSConfig:** Troubleshoots startup issues and manages boot options, services, and system tools.
- **Advanced System Settings:** Configure virtual memory (page file) and crash dump settings.
- **UAC (User Account Control):** Prevents unauthorized system changes by prompting for elevated privileges.
- **Computer Management:** Central console for Task Scheduler, Event Viewer, Device Manager, Disk Management, Services, and Performance Monitor.
- **Event Viewer:** Stores Windows logs used for troubleshooting and security investigations.
- **Task Scheduler:** Automates tasks based on schedules or system events.
- **System Information (`msinfo32`):** Displays hardware, software, environment variables, and system configuration.
- **Resource Monitor (`resmon`):** Monitors CPU, memory, disk, and network usage in real time.
- **Common Command Prompt Commands:**
  - `hostname` – Display computer name.
  - `whoami` – Display current user.
  - `ipconfig` – View network configuration.
  - `netstat` – View network connections.
  - `net` – Manage network resources.
- **Windows Registry (`regedit`):** Central database for Windows configuration; improper changes can impact system stability.
